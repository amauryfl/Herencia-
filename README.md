# Actividad de Herencia

## ¿Qué es la herencia?

La herencia es un concepto de la programación orientada a objetos que permite que una clase herede atributos y métodos de otra clase. Esto facilita la reutilización del código, mejora la organización del programa y hace que el desarrollo sea más claro y mantenible.

## Tipos de herencia

### 1. Herencia simple
Sucede cuando una clase hereda de una sola clase base.

### 2. Herencia jerárquica
Se presenta cuando varias clases heredan de una misma clase padre.

### 3. Herencia multinivel
Ocurre cuando una clase hereda de otra clase que a su vez heredó de una clase base.

### 4. Herencia múltiple
Es cuando una clase hereda de varias clases al mismo tiempo. En C# no se permite directamente con clases, pero puede simularse con interfaces.

## Ejemplos incluidos

1. Persona y Estudiante
2. Vehículo y Carro
3. Animal y Perro
4. Empleado y Gerente
5. Producto y Bebida

## Conclusión

La herencia permite reutilizar código, evitar repeticiones y representar relaciones entre objetos de forma más lógica. Gracias a este principio, los programas pueden ser más organizados, flexibles y fáciles de mantener.
